# 이펙티브 c#

- 이펙티브 C# (3판) 내용 정리

## 챕터1 - C# 언어 요소

* 아이템 01 : 지역변수를 선언할 때는 var를 사용하는 것이 낫다.
* 아이템 02 : const보다는 readonly 가 좋다.
* 아이템 03 : 캐스트보다는 is, as가 좋다.
* 아이템 04 : string.Format()을 보간 문자열로 대체하라.
* 아이템 05 : 문화권별로 다른 문자열을 생성하려면 FormattableString을 사용하라.
* 아이템 06 : nameof() 연산자를 적극 활용하라.
* 아이템 07 : 델리게이트를 이용하여 콜백을 표현하라.
* 아이템 08 : 이벤트 호출 시에는 null 조건 연산자를 사용하라.
* 아이템 09 : 박싱과 언박싱을 최소화하라.
* 아이템 10 : 베이스 클래스가 업그레이드된 경우에만 new 한정자를 사용하라.
